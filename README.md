# xhxt
基于 SpringBoot+Vue 的前后端分离鲜花商城与社区管理系统，包含管理员、员工、用户三级权限，实现商品管理、订单履约、社区互动、内容运营等完整电商业务。

## 系统角色
管理员：系统最高权限，负责用户管理、商品管理、内容管理、社区管控、数据统计、系统配置
员工：负责订单处理、库存维护、客服回复、日常运营，无系统全局配置权限
普通用户：商城购物、订单管理、社区发帖评论、收藏互动

## 核心功能
### 一、管理员功能模块
用户与权限管理：管理员/员工/会员账号的新增、编辑、删除、查询；违规用户封禁/删除

商品中心管理：商品分类维护、商品信息管理（新增/编辑/上下架）

内容与社区管理：轮播图、系统公告、新闻资讯维护；论坛板块管理、帖子审核/置顶/删除

财务与数据面板：全站销售额、订单量等宏观数据统计

个人中心：管理员自身资料、密码修改

### 二、员工功能模块
商品与库存维护：库存查询、库存数量调整；商品信息浏览

订单与履约处理：全量订单状态查看；待发货订单物流录入、状态更新

社区与客服互动：以官方身份回复用户评论/提问

个人中心：员工自身资料、密码修改

### 三、用户功能模块
账户与通行证：注册登录、个人资料维护、收货地址管理

商城购物体验：商品浏览/搜索、商品详情查看、收藏/加购/下单、购物车结算、订单物流查询

内容与社区互动：新闻资讯浏览/点赞/收藏/评论；论坛发帖/互动、我的收藏管理

## 技术栈
后端：SpringBoot、MyBatis-Plus、MySQL
前端：Vue、ElementUI
架构：前后端分离、RBAC权限控制、RESTful API

## 适用场景
毕业设计、课程设计、电商实战项目、小型鲜花商城系统
